# Discovering Community Tools

## When to use this
Use when looking for pre-built community ETL connectors (Kafka, S3), visualization dashboard plugins, or ORM adapters.

## Prerequisites
Access to GitHub. Links:
- [TigerGraph Ecosystem Hub](https://github.com/tigergraph/ecosys/blob/master/docs/awesome.md)

## Steps
1. Navigate to the official [TigerGraph Ecosystem Awesome List](https://github.com/tigergraph/ecosys/blob/master/docs/awesome.md) on GitHub.
2. Browse categories: IDE extensions, connectors, loaders, frameworks, and visualization plugins.
3. Verify project activity (recent commits, release tags) before integrating community tools into enterprise pipelines.

## Common mistakes
- Utilizing unmaintained community libraries that contain outdated GSQL APIs and dependencies without testing locally first.
- Confusing third-party community tools with officially supported TigerGraph products.
