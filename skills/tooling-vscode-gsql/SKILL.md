# VS Code GSQL Extension Setup

## When to use this
Use when developers want a smooth IDE experience when writing GSQL queries outside of GraphStudio.

## Prerequisites
VS Code installed. Links:
- [TigerGraph Developer Tools](https://github.com/tigergraph/ecosys/blob/master/docs/awesome.md)

## Steps
1. Open VS Code extension marketplace.
2. Search for the official **GSQL** or **TigerGraph GSQL** syntax highlighting extension.
3. Click **Install**.
4. Create a folder in your workspace, save files with the `.gsql` extension, and check that keywords and comments highlight correctly.

## Common mistakes
- Confusing standard SQL extensions with GSQL extensions. Standard SQL formatters will distort GSQL accumulator syntaxes.
