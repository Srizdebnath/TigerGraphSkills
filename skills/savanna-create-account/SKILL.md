# Creating a TigerGraph Savanna Account

## When to use this
Use when a user wants to set up a new TigerGraph Savanna account, access the cloud console, or begin their TigerGraph Cloud journey.

## Prerequisites
An email address, web browser, and internet connection. Links:
- [TigerGraph Savanna Portal](https://savanna.tigergraph.com)
- [Savanna Overview Documentation](https://docs.tigergraph.com/savanna/main/overview/overview)

## Steps
1. Navigate to the [TigerGraph Savanna Sign-up Page](https://savanna.tigergraph.com).
2. Register using email and password, or authenticate via Google/GitHub single sign-on (SSO).
3. Complete email verification.
4. Log in to the TigerGraph Savanna console to access your default workgroup.

## Common mistakes
- Conflating Savanna with TigerGraph Cloud Classic (legacy console at tgcloud.io). Always use savanna.tigergraph.com for new deployments.
- Assuming credit card details are required immediately. Free tier accounts can be created without entering payment information.
